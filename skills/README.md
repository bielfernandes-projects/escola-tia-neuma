Nós iremos fazer um site institucional com o intuito de ser um template para apresentar para possíveis clientes donos de escola infantil. Então precisamos focar em um site atrativo para que o cliente olhe esse template e imagine que seja o seu site e assim, feche negócio comigo para o criar o seu próprio site.

Para iniciarmos nosso projeto, quero que você analise a pasta contida nele que contém as skills e agentes e carregue as skills relevantes para sua tarefa.

Lembrando que vou em breve te mandar as instruções para construirmos esse site e aí sim, você entende e escolhe os agentes/skills necessários para a tarefa.
